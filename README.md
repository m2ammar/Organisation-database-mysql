# Organisation Database (MySQL)

## Description
A MySQL relational database for managing an organisation’s data.  
Includes tables for students, courses, employees, departments, projects, teachers, and workers, with relationships, primary keys, and foreign keys.

## Tables
- `course`, `department`, `DEPT`, `employees`, `projects`, `student`, `TEACHER`, `worker`

## Features
- Relational tables with primary and foreign keys
- Sample data included for testing
- Supports queries across multiple tables

## Tools
- MySQL 
- MySQL Workbench
